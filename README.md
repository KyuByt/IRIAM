# IRIAM
An IRIAM themed vanilla+ model as a gift on their first anniversary!

# How to use the model
  1. Download Minecraft Forge or Fabric
  2. Download [Figura](https://modrinth.com/mod/figura/) for the respective version
  3. Add the "Figura" mod to the mods folder
  4. Open a new world (singleplayer or multiplayer) and click the ESC key once it's done loading
  5. Click the Figura button besides "Open to LAN"
  6. Click the folder icon in the top left
  7. Paste the .zip file inside there
  8. Select the IRIAM model for the respective version
